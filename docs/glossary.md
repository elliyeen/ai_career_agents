# Glossary

CareerOS — v0.1

---

## Agent

A discrete, stateless workflow unit. Each agent has a defined mission, inputs, outputs, rules, and quality checklist. Agents are defined in Markdown (Phase 1) and implemented in Rust (Phase 2). Agents never call each other directly.

---

## Application

A record linking a specific Job to a specific Resume, with a tracked status across the pipeline (draft → submitted → responded → interview → offer).

---

## Artifact

A file produced by an agent and written to `outputs/`. Examples: a tailored resume, an outreach draft, an interview prep document, a weekly review report.

---

## ATS

Applicant Tracking System. Software used by employers to parse and filter resumes before human review. Resumes must be plain text, keyword-rich, and free of complex formatting to pass ATS filters.

---

## Candidate Profile

A Markdown file in `docs/memory/` containing the user's verified skills, target roles, location preferences, compensation requirements, and constraints. The ground truth for qualification scoring.

---

## Conversion Rate

The percentage of items that advance from one pipeline stage to the next. Example: Applications → Responses. Measured weekly by the Retrospective agent.

---

## Deduplication

The process of checking a newly discovered job URL against existing records and discarding it if already stored. Prevents duplicate processing.

---

## Event

An immutable record written to the `events` table when something meaningful happens. Events are the only signal between agents. They are never updated or deleted.

---

## Experience

Verified work history, skills, and measurable outcomes stored in `docs/memory/experience.md`. The only permitted source of content for resume generation. Nothing may be added to a resume that is not in this file.

---

## Human Approval Gate

A required pause before any external action. The system drafts output, emits an event, and waits. A human reviews the artifact and issues an approval command via CLI before execution continues.

---

## Job

A discovered job posting. Stored with URL, title, company, description, location, and qualification status.

---

## Master Resume

The base resume document stored in `docs/memory/resume-master.md`. Contains all verified experience sections. The Resume agent selects and tailors relevant sections for each job — it never adds content that is not in the master.

---

## Memory

The set of Markdown files in `docs/memory/` that define who the candidate is: experience, skills, profile, criteria. Memory is the system's source of truth for all content generation.

---

## Outreach

Recruiter-directed communication drafted by the Outreach agent. Types: initial message, follow-up, thank-you note. All outreach requires human approval before sending.

---

## Pipeline

The full sequence of stages a job opportunity moves through: Discovery → Qualification → Resume → Outreach → Tracking → Interview → Offer → Retrospective.

---

## Qualification

The process of evaluating a discovered job against the candidate profile and scoring it as qualified or disqualified. Disqualified jobs record a reason. They are never silently dropped.

---

## Resume

A tailored, ATS-friendly document generated for a specific job. Based only on verified experience. Versioned. Must be human-approved before use.

---

## Retrospective

The weekly agent that measures pipeline conversion rates, identifies the weakest stage, and produces written recommendations. The primary mechanism for system improvement over time.

---

## STAR Story

A structured behavioral interview response format: Situation, Task, Action, Result. Generated by the Interview Prep agent from verified experience.

---

## State Machine

The defined set of valid application statuses and the transitions between them. Enforced by the system. Invalid transitions are rejected.

---

## Verified Experience

Work history, skills, metrics, and accomplishments that are confirmed true and stored in `docs/memory/experience.md`. The only permitted basis for any content the system generates. No fabrication of any kind is permitted.
