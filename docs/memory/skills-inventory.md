# Skills Inventory

Every skill listed here must be demonstrable.
"Demonstrable" means: you can answer a technical question, complete a task, or name a project where you used it.

Agents use this file to score job fit and avoid claiming skills you do not have.

---

## Skill Levels

- **Expert** — You could teach this. You have used it for 3+ years on real projects.
- **Proficient** — You can work independently. You have used it on 2+ real projects.
- **Familiar** — You understand concepts. You have used it with guidance or in small projects.
- **Learning** — You are actively studying. Do not claim this on resumes yet.

## Notes for Agents

- If a job requires a skill not in this file, flag it: "Skill not in inventory — human review required."
