## Negotiation Rules for Agents

- Never state a number first in outreach. Express range only when directly asked.
- Use this phrase when asked: "I'm targeting a total comp of $[TARGET] — open to discussing the mix."
- Never accept verbally in outreach. All approvals require human confirmation.
